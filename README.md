- ✅ Sudoku

## ⚡ How to play the game

Open your desired terminal
    
    cd <folder>

To download the game:

    git clone https://github.com/AbbeyIT/game-hub.git
    
To navigate the game:

    cd game-hub
    
To play the game:

    python main.py
